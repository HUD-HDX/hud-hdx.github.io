Guide to Using HDX Summary Report Templates

The SPM and HIC modules have provided Excel-based summary report templates. These templates are available on the "SPM Reports", "PIT Reports", and "HIC Reports" pages of each module.
To produce a given summary report:
  1. Go to the Reports page for the module in question (e.g. modules/hudhdx2.info/spm/reports) and download the raw data .csv file for a single selected CoC. Open this .csv file in Excel.
  2. Download the Summary Report Template (an .xlsx file). If prompted, click "Enable Editing".
  3. Copy the data from the .csv file.
    1. For the SPM you will copy two rows: a header row and a row of data.
    2. For the PIT, you will copy three rows: a row of header variable names, a row of header plain language descriptions, and a row of data.
    3. For the HIC, you will copy only the data rows (not the headers).
  4. Paste the data from the .csv file into the raw data tab of the .xlsx file. Refer to the screenshots below for reference.
